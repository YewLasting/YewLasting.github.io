1122. 数组的相对排序

难度：简单

题干

给你两个数组，arr1 和 arr2，

  • arr2 中的元素各不相同
  • arr2 中的每个元素都出现在 arr1 中

对 arr1 中的元素进行排序，使 arr1 中项的相对顺序和 arr2 中的相对顺序相同。未在 arr2 中出现过的元素需要按照升序放在 arr1 的末尾。

示例：

输入：arr1 = [2,3,1,3,2,4,6,7,9,2,19], arr2 = [2,1,4,3,9,6]
输出：[2,2,2,1,4,3,3,9,6,7,19]

提示：

  • arr1.length, arr2.length <= 1000
  • 0 <= arr1[i], arr2[i] <= 1000
  • arr2 中的元素 arr2[i] 各不相同
  • arr2 中的每个元素 arr2[i] 都出现在 arr1 中

来源：力扣（LeetCode）

链接：https://leetcode-cn.com/problems/relative-sort-array/

题解

语言：

C++

// 2020.11.14
#include <iostream>
#include <vector>
#include <queue>
#include <stack>
#include <cstring>
#include <algorithm>
#include <unordered_map>
#include <unordered_set>
using namespace std;
class Solution {
public:
    vector<int> relativeSortArray(vector<int>& arr1, vector<int>& arr2) {
        int upper = *max_element(arr1.begin(), arr1.end());
        vector<int> frequency(upper + 1);//记录arr1中每个数字出现的次数，加1是因为数组索引从0开始，不加1无法表示frequency[upper]
        for (int x: arr1) {
            ++frequency[x];
        }
        vector<int> ans;
        for (int x: arr2) {//遍历arr2,把每个元素按顺序添加进去
            for (int i = 0; i < frequency[x]; ++i) {//根据出现次数，添加相应个数
                ans.push_back(x);
            }
            frequency[x] = 0;//添加完毕后置为零作为标记
        }
        for (int x = 0; x <= upper; ++x) {//arr2中的数据添加完毕后从小到大添加剩余的个数
            for (int i = 0; i < frequency[x]; ++i) {//只有frequency[x]>0的数据才会被添加
                ans.push_back(x);
            }
        }
        return ans;
    }
};
//进行测试，输出测试结果
int main()
{
    Solution test;
    vector<int> arr1={2,3,1,3,2,4,6,7,9,2,19};
    vector<int> arr2={2,1,4,3,9,6};
    vector<int> ans = test.relativeSortArray(arr1, arr2);
    for (int i : ans)
        cout << i << " ";
    cout << endl;
    getchar(); //获取一个字符，按键盘任意键即可，可以使得输出结果暂停在屏幕上
    return 0;
}
